Listed elements 指 form.elements 和 fieldset.elements 包含的 elements。这些 elements 也可以指定 form attribute。

Submittable elements，，在提交表单时，这些 elements 的 value 构成表单数据。

Resettable elements，这些 elements 可以重置表单。

labelable elements，这些 elements 可以可以使用 label， 部分 elements 不是表单相关 elements。
